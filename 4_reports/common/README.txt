This folder contains the word-styles-reference document and a Zotero citation style (.csl) file for writing mansucripts in R Markdown 

word-styles-reference.docx: a basic word styles reference document that is called from the R Markdown file 

ecology.csl: a citation style file that formats the bibliography as per the Ecological Society of America formatting style in the journal Ecology. 
	.csl files for other journals can be downloaded here: https://www.zotero.org/styles
